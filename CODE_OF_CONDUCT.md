# Code of Conduct

## Our Goal

We want contribution to this project to be respectful, constructive, and welcoming for people with different levels of
experience.

## Expected Behavior

- communicate respectfully
- discuss code and ideas, not people
- explain technical feedback clearly
- help newcomers understand the project
- return to facts when discussions become tense

## Unacceptable Behavior

- insults or harassment
- personal attacks
- discrimination
- intentional disruption of issues or pull requests
- sharing private information without permission

## Enforcement

Project maintainers may moderate discussions, reject contributions, or restrict participation when these rules are
violated.

## Scope

This code of conduct applies to issues, pull requests, reviews, comments, and other project-related discussions.
